# Ugly Numbers
##  Hard 
<div class="problem-statement">
                <p></p><p><span style="font-size:18px">Ugly numbers are numbers whose <strong>only prime factors are 2, 3 or 5</strong>. The sequence 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, … shows the first 11 ugly numbers. By convention, 1 is included. Write a program to find <strong>N<sup>th</sup></strong> Ugly Number.</span></p>

<p><span style="font-size:18px"><strong>Example 1:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:
</strong>N = 10
<strong>Output: </strong>12
<strong>Explanation:</strong> 10<sub>th</sub>&nbsp;ugly number is 12.</span></pre>

<p><span style="font-size:18px"><strong>Example 2:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:
</strong>N = 4
<strong>Output:</strong> 4
<strong>Explanation: </strong>4<sub>th</sub>&nbsp;ugly number is 4.</span></pre>

<p><span style="font-size:18px"><strong>Your Task:</strong><br>
You don't need to read input or print anything. Your task is to complete the function&nbsp;<strong>getNthUglyNo()</strong>&nbsp;which takes an integer&nbsp;<strong>n</strong><strong>&nbsp;</strong>as parameters and returns an integer denoting the answer.</span></p>

<p><span style="font-size:18px"><strong>Expected Time Complexity:</strong>&nbsp;O(N)<br>
<strong>Expected Auxiliary Space:</strong>&nbsp;O(N)</span></p>

<p><span style="font-size:18px"><strong>Constraints:</strong><br>
1 ≤ N ≤ 10<sup>4</sup></span></p>
